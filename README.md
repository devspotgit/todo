# Todo

Classic TODO app - save and manage your tasks.

![](https://i.ibb.co/BKGMB3BR/todo.png)

## Features

- User authentication
- Responsive and adaptable even on mobile
- Add, rename, remove tasks
- Set task status

## Technologies

- Vue js as frontend framework
- Express to manage api
- MongoDB to save user tasks
